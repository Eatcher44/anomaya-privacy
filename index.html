Dernière mise à jour : 09/03/2026 Anomaya respecte votre vie privée et s’engage à protéger les données personnelles de ses utilisateurs. Cette politique de confidentialité explique quelles données sont collectées lorsque vous utilisez l’application Anomaya et comment elles sont utilisées. 1. Données collectées Lorsque vous utilisez Anomaya, nous pouvons collecter les informations suivantes : Informations de compte - adresse email - identifiant utilisateur Données liées aux animaux - nom - espèce - race - sexe - date de naissance - poids - données de santé (vaccins, vermifuge, soins) - informations d’élevage (portées, parents, généalogie) Photos - photos d’animaux ajoutées par l’utilisateur Ces données sont fournies directement par l’utilisateur. 2. Utilisation des données Les données collectées sont utilisées uniquement pour : - fournir les fonctionnalités de l’application - gérer les profils d’animaux - suivre les informations de santé - gérer les portées et l’élevage - permettre le partage de fiches animaux - améliorer l’expérience utilisateur 3. Publicités L’application peut afficher des publicités via des services tiers. Ces services publicitaires peuvent utiliser certaines informations techniques (comme le type d’appareil ou un identifiant publicitaire) afin d’afficher des annonces pertinentes. Les utilisateurs peuvent supprimer les publicités en souscrivant à l’abonnement proposé dans l’application. 4. Abonnements et paiements L’application peut proposer des abonnements payants. Les paiements sont gérés par les plateformes de distribution (Google Play ou Apple App Store). Anomaya ne collecte pas directement les informations de paiement telles que les numéros de carte bancaire. 5. Partage des données Anomaya ne vend ni ne partage les données personnelles des utilisateurs à des tiers. Certaines données peuvent être accessibles publiquement uniquement lorsque l’utilisateur choisit de partager une fiche animal via un lien public. 6. Stockage des données Les données sont stockées sur des serveurs sécurisés afin de fournir les fonctionnalités de l’application. Nous mettons en place des mesures raisonnables pour protéger les informations contre l’accès non autorisé. 7. Suppression des données Les utilisateurs peuvent supprimer leur compte directement depuis l’application. La suppression du compte entraîne la suppression des données associées : - profil utilisateur - animaux - portées - photos - historiques 8. Sécurité Nous mettons en œuvre des mesures de sécurité techniques pour protéger les données contre la perte, l’accès non autorisé ou la modification. 9. Modifications Cette politique de confidentialité peut être mise à jour afin de refléter les évolutions de l’application. 10. Contact Pour toute question concernant cette politique de confidentialité, vous pouvez nous contacter : support@anomaya.app
